# 🚀 Java Data Structures & Algorithms Repository

## 📖 About This Repository

This repository is a curated collection of **Data Structures and Algorithms (DSA)** implementations in **Java**, developed with a focus on:

* Writing clean, maintainable, and efficient code
* Understanding core algorithmic patterns
* Preparing for technical interviews (TCS, product-based companies)
* Strengthening backend problem-solving skills

The repository follows a **problem-driven approach**, where each implementation is designed with clarity, correctness, and optimal complexity in mind.

---

## 🎯 Objectives

* Build strong fundamentals in DSA
* Practice industry-relevant problem-solving techniques
* Improve time and space complexity optimization
* Develop coding patterns useful in real-world backend systems

---

## 🧩 Implemented Problems

### 🔹 Array-Based Problems

#### 1. Find Minimum, Maximum, Second Minimum, and Second Maximum

* Single traversal solution
* Optimized comparison logic
* Handles edge cases (duplicates, small arrays)

---

#### 2. Check if an Array is Sorted

* Linear time complexity **O(n)**
* Early termination for efficiency

---

#### 3. Remove Duplicates from Sorted Array

* Two-pointer technique
* In-place modification (**O(1) space**)

---

#### 4. Maximize Number After Removing a Digit

* Greedy + string manipulation approach
* Demonstrates decision-making optimization
